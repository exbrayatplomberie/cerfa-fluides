PROCÉDURE GITHUB

1. Décompressez le ZIP.
2. Ouvrez le fichier app.js contenu dans le dossier.

3. Sur GitHub, ouvrez le dépôt cerfa-fluides.
4. Ouvrez app.js.
5. Cliquez sur le crayon Edit.
6. Cliquez dans le texte et faites Ctrl + A.
7. Ouvrez le nouveau app.js, faites Ctrl + A puis Ctrl + C.
8. Revenez sur GitHub et faites Ctrl + V.
9. Cliquez sur Commit changes.
10. Message conseillé :
    Correction séparation observations signatures

VÉRIFICATION IMPORTANTE DANS INDEX.HTML

11. Ouvrez index.html.
12. Vérifiez qu'il n'existe PLUS de ligne :
    <script src="rapport-2-pages.js?..."></script>
    Si elle existe encore, supprimez-la.

13. Remplacez la ligne app.js par :
    <script src="app.js?v=0.4.3.1"></script>

14. Cliquez sur Commit changes.
15. Attendez 2 minutes.
16. Sur PC : fermez l'application, rouvrez-la puis faites Ctrl + F5.
17. Sur iPad : fermez complètement l'application puis rouvrez-la.

Pense à faire une sauvegarde régulièrement. Les données sont stockées dans Safari sur l'iPad.

Installation sur iPad

  1. Ouvre l'application avec Safari.
  2. Touche le bouton Partager (carré avec flèche vers le haut).
  3. Choisis « Ajouter à l'écran d'accueil ».
  4. Nom : « CERFA Fluides » puis « Ajouter ».

Créer et enregistrer un PDF

  1. Remplis la fiche et touche « Créer le PDF ».
  2. Dans l'aperçu, touche Partager puis « Enregistrer dans Fichiers », ou « Imprimer ».
  3. Pour obtenir un PDF depuis l'écran d'impression, écarte deux doigts sur l'aperçu puis touche Partager.

Important : le bouton « CERFA officiel + attestation » remplit le CERFA 15497*04 original et ajoute en deuxième page l’attestation de capacité SOCOTEC. Vérifie les informations avant archivage.

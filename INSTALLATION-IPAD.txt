MISE A JOUR GITHUB
1. Ouvrir le dépôt cerfa-fluides.
2. Add file > Upload files.
3. Déposer TOUS les fichiers de ce dossier et confirmer le remplacement.
4. Commit changes.
5. Attendre 1 à 3 minutes, puis recharger l'application.
6. Sur iPad, supprimer l'ancienne icône puis ajouter à nouveau le site à l'écran d'accueil si la mise à jour ne s'affiche pas.
